# 云打包

使用 HBuilderX 进行云打包时，需要使用 DCloud 账号登录。[查看如何注册账号](../account/reg.md)

## 打包模式

HBuilderX 支持两种打包模式：

| 模式 | 说明 |
| --- | --- |
| 传统打包 | 向 DCloud 服务器上传代码及证书，DCloud 承诺不保留相关资源，打包完成后自动清理 |
| 安心打包 | 不上传代码及证书，打包速度更快。[查看详情](https://uniapp.dcloud.net.cn/tutorial/build/SafePack.html#%E5%AE%89%E5%BF%83%E6%89%93%E5%8C%85%E4%BD%BF%E7%94%A8%E6%8C%87%E5%8D%97) |

> 推荐优先使用安心打包。

## 打包配额

### 打包次数

DCloud 为每个开发者账号提供了充足的免费打包额度。为避免资源滥用影响其他用户，系统会按天、按月限制账号的打包次数（风控规则不对外公开）。

如遇打包次数不足，可登录开发者中心，在 [打包个数充值](https://dev.dcloud.net.cn/pages/product-account/product-account?pcd=app_package_quantity_recharge) 页面进行充值。

**计费说明：**

- 付费打包每次 2 元，仅限充值账号使用，不可转让
- 优先扣减免费次数，免费额度不足时才扣减付费次数
- 打包任务成功提交到服务器即扣费，无论最终打包成功或失败

**省钱技巧：**

- 开发调试阶段建议使用内置基座或自定义基座，减少打包次数消耗
- 推荐优先使用安心打包

### App 大小超限

部分 App 因静态资源或三方插件较多，可能导致 wgt 包体积较大，消耗更多服务器资源，此时需付费打包。

**计费规则：**

1. **计费依据**：按提交的 wgt 包体积计费。wgt 包是上传到云打包服务器的资源文件，包含前端代码、splash、icon 图、本地原生插件等，不是最终的 APK 或 IPA。

2. **收费标准**：
   - 40MB—100MB：每次 10 元
   - 每增加 100MB：费用增加 10 元
   - 充值金额较高时可享优惠折扣（[详见充值页面](https://dev.dcloud.net.cn/pages/product-account/product-account?pcd=app_size_limit_recharge)）

3. **多平台计费**：Android 和 iOS 分别计费。同时勾选两个平台则扣费两次；Android 勾选多个渠道包时，每个渠道包单独计费。
   > 示例：wgt 体积 40MB 时，单平台打包收费 10 元，双平台打包收费 20 元。

4. **插件开发者**：为插件市场提供插件做测试时仍需付费，但插件审核通过后，相关打包费用将退还至插件市场收入，可在次月「插件市场 -> 我的插件 -> 付款」页面查看。

5. **iOS 符号表**：勾选「生成 iOS 符号表（dsym）文件」将额外收取 10 元。

6. **其他说明**：
   - 充值额度仅限当前账号使用，不可转让
   - 打包任务成功提交即扣费，无论最终打包成功或失败

> 大小超限计费规则将于 2026 年初调整，详见 [DCloud 云打包计费规则调整公告](https://ask.dcloud.net.cn/article/41541)

### 打包排队

DCloud 提供免费打包服务器，但随着 Android、iOS 系统升级，编译工具对硬件资源需求增加，高峰期可能出现排队。如需减少等待时间，可选择以下方案：

#### 独享打包机

适用于打包不频繁、临时有加速需求的用户。

- **计费方式**：按次计费
- **工作原理**：启用后启动专属打包服务器，打包结束后自动销毁
- **使用前提**：账号需有足够余额。[查看余额](https://dev.dcloud.net.cn/pages/product-account/product-account?pcd=app_pay_build_queue)

**启用方式：**

- 打包高峰期提交打包后，HBuilderX 控制台会输出启用链接，点击即可快速启用
- 或登录开发者中心 [打包队列](https://dev.dcloud.net.cn/pages/cloudbuild/qlist) 页面，选择排队任务进行加速

#### 企业打包机

企业独享打包，无需排队。适用于企业用户及打包频繁的个人用户。[咨询开通](https://im.dcloud.net.cn/#/?business_type=enterprise-cloud-build)

**专属权益：**

| 权益 | 说明 |
| --- | --- |
| 免费打包额度 | 每年 2 万次 |
| wgt 体积免费额度 | 单次 200MB 以内免费 |
| iOS 符号表 | 免费 |

超出 2 万次或 200MB 的部分另行收费。

**协作者规则：**

- 开通账号需通过企业实名认证，且绑定企业邮箱
- 支持最多 9 个协作者，协作者邮箱需与项目创建者为同一企业邮箱
- 如需添加更多协作者，请联系客服

[查看如何进行企业认证](https://uniapp.dcloud.net.cn/dev/account/real-name-verification.html)
