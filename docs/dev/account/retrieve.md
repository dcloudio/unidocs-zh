# 找回账号

当您因员工离职、忘记账号关联的邮箱或手机号码等原因无法登录时,可以通过身份认证的方式找回账号。

## 功能说明

找回账号功能支持通过个人身份认证或企业信息认证来找回您的登录信息。系统会根据您提供的认证信息,在通过实人认证后显示账号关联的邮箱和手机号,您还可以选择重置密码、修改邮箱或手机号。



## 找回流程

### 1. 个人账号找回流程

个人实名认证账号可通过个人身份认证找回,流程如下:

1. 点击[“找回账号”页面](https://account.dcloud.net.cn/uni_modules/uni-id-pages/pages/retrieve/retrieve-username)
2. 选择"个人身份"标签
3. 填写实名信息:
   - **真实姓名**: 必须与账号实名认证时填写的姓名完全一致
   - **身份证号**: 支持15位或18位身份证号码,必须与实名认证时填写的身份证号一致
4. 点击"下一步"进行身份验证
5. 使用手机扫描页面显示的二维码,完成实人扫脸认证
6. 点击"我已完成认证"按钮,系统会验证认证状态
7. 认证通过后,系统将显示账号关联的邮箱和手机号

### 2. 企业账号找回流程

`由于实名认证时未保存法人的身份证号码，因此如需找回账号，请提前将法人身份证正反面照片发送至邮箱 `service@dcloud.io`，并在邮件中详细描述您的问题。收到官方回复后，再由法人参考第5步进行操作。`

企业实名认证账号可通过企业信息认证找回,流程如下:
1. 点击[“找回账号”页面](https://account.dcloud.net.cn/uni_modules/uni-id-pages/pages/retrieve/retrieve-username)
2. 选择"企业信息"标签
3. 填写企业信息:
   - **企业名称**: 必须与账号企业实名认证时填写的名称完全一致
   - **统一社会信用代码**: 必须为有效的18位统一社会信用代码,且与实名认证时填写的代码一致
4. 点击"下一步"进行企业信息验证
5. 使用手机扫描页面显示的二维码,完成实人扫脸认证(需法人操作)
6. 点击"我已完成认证"按钮,系统会验证认证状态
7. 认证通过后,系统将显示账号关联的邮箱和手机号
