# uni-group 分组

::: tip 组件名：uni-group
> 代码块： `uGroup`

[点击下载&安装](https://ext.dcloud.net.cn/plugin?name=uni-group)
:::

分组组件可用于将组件分组，添加间隔，以产生明显的区块。

## 介绍
### 基本用法


```html
<uni-group title="分组1" top="20">
    <view>分组1 的内容</view>
    <view>分组1 的内容</view>
</uni-group>

<uni-group title="分组2">
    <view>分组2 的内容</view>
    <view>分组2 的内容</view>
</uni-group>
```
## API
### Group Props

|属性名|类型|默认值|说明|
|:-:|:-:|:-:|:-:|
|title|String|-|主标题|
|top|Number|-|分组间隔|
|mode|String|''|模式 ，card 为卡片模式|
