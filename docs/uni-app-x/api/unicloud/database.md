## databaseForJQL() @databaseforjql

**和uni-app项目接口差异**

- uni-app x项目内数据库的客户端接口仅支持databaseForJQL不再支持database方法。
- action云函数因为安全问题已经不再推荐使用。开发者应使用[数据库触发器](https://uniapp.dcloud.net.cn/uniCloud/jql-schema-ext.html)来实现相关功能。
- 暂不支持multiSend
