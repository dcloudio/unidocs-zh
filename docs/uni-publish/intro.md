# uni多商店上传

## 介绍@intro

一键发布到多家应用商店，高效助力App发版。

当前支持的应用商店：**App Store、Google Play、华为、荣耀、小米、VIVO、OPPO、应用宝、魅族**。  

**上架 App Store 使用 HBuilderX 云打包 + uni多商店上传 即可完成全流程上架，全程无需 Mac 电脑。[应用首次上架使用教程](./first.md)**

> 欢迎进入[uni多商店上传技术交流群](https://im.dcloud.net.cn/#/?joinGroup=68d10a5f46d4d0b06752ddfb)



## 产品优势@advantage
**1. 一键多发**

支持一键将应用发布到多家应用商店，大幅提升发版效率。


**2. 安全稳定**

通过各大应用市场官方Api上传应用，无需手动登录，安全、稳定、快捷。

**3. 审核状态实时更新**

App 提交成功后，可实时查看在各大应用商店的审核状态。

**4. 渠道包灵活分发**

在提交应用商店时，如果您需要对每个应用商店统计不同的数据来源，您可以选择分发不同的渠道包。

**5. 免 Mac 完成 iOS 全流程上架**

搭配 HBuilderX 云打包使用，从打包、App Store 应用创建到最终上架全流程无需 Mac 电脑，跨平台即可完成 App Store 发布，降低 iOS 上架硬件门槛。
