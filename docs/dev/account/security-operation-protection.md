# 操作保护

## 操作场景

当您在控制台执行敏感操作时，开启操作保护可以在操作前再增加一次身份校验，确保操作者确为本人，降低账号被冒用的风险。本文介绍操作保护的类型，以及如何开启/关闭操作保护的步骤。

> 说明：敏感操作校验有效期为 15 分钟，如果您超过 15 分钟再次进行敏感操作则需再次验证。


## 操作保护类型

| 操作保护类型 | 作用 |
| --- | --- |
| 开启微信扫码验证 | 在控制台进行相应操作时，进入身份验证页面，需要用微信扫码确认身份。扫码与账号关联微信一致后才能继续操作。 |
| 输入登录密码进行确认 | 在控制台进行相关操作时，需要输入登录密码。密码输入正确后才能继续执行。 |
| 不开启 | 不进行二次认证。 |

## 操作步骤

### 开启操作保护

1. 登录[DCloud 开发者中心](https://dev.dcloud.net.cn/)，进入 `个人中心` -  `账户信息` 页面。
2. 在 `账号保护` 栏中，找到 `操作保护`，点击 `编辑`。
3. 在弹出的窗口中，根据实际需求选择操作保护方式，点击 `确定`，即可完成设置。

![image](https://web-ext-storage.dcloud.net.cn/doc/dev/protect_list.png)

![image](https://web-ext-storage.dcloud.net.cn/doc/dev/protect_add202512011500.png)
### 关闭操作保护

1. 登录[DCloud 开发者中心](https://dev.dcloud.net.cn/)，进入 `个人中心` -  `账户信息` 页面。
2. 在 `账号保护` 栏中，找到 `操作保护`，点击 `编辑`。
3. 在弹出的窗口中，根据实际需求选择 `不开启`，点击 `确定`，即可完成设置。

![image](https://web-ext-storage.dcloud.net.cn/doc/dev/protect_add202512011501.png)
